# Simple LAN File Share (SLFS)

同一ネットワーク（Wi-Fiなど）内のデバイス間で、ブラウザを通して素早くファイルを共有するためのPythonスクリプトです。
送信側がファイルをアップロードすると、受信側がQRコードやURLから即座にダウンロードできるようになります。

## ✨ 特徴
- **インストール不要**: PythonとFlaskがあればすぐに動かせます。
- **QRコード自動生成**: 受信側のスマホなどでURLを打ち込む手間がありません。
- **デュアルポート設計**: 
  - 送信管理画面（自分用）: `localhost:5001`
  - 公開ダウンロード画面（相手用）: `IPアドレス:5000`

## 🛠 準備
実行には `Flask` が必要です。インストールしていない場合は以下のコマンドで追加してください。

```bash
pip install flask
```

## 🚀 使い方
1. スクリプトを実行する
    ```Bash
    python main.py
    ```
2. 送信画面が開く
    実行すると自動的にブラウザで http://localhost:5001 が開きます。

3. ファイルをアップロード
    「ファイルを選択」から共有したいファイルを選びます。

4. 相手に共有
    画面に表示される QRコード または URL を相手に教えます。

5. 相手がダウンロード
    相手がURLにアクセスすると、ダウンロードボタンが表示されます。

## 🌐 ポート
* Admin App (Port 5001): 自分のPCからのみアクセス可能。ファイルの選択と状態管理を行います。
* Public App (Port 5000): ネットワーク内の全員がアクセス可能。ファイルのダウンロード専用画面です。

## ⚠️ 注意事項
* セキュリティ: このスクリプトには認証機能がありません。信頼できるローカルネットワーク内でのみ使用してください。
* ファイアウォール: 相手がアクセスできない場合は、PCのファイアウォール設定でポート 5000 が許可されているか確認してください。
